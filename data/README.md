# Data Resources

This directory provides access to datasets generated in this study.

## T2T genome assemblies

The phased telomere-to-telomere (T2T) genome assemblies for the pedigree individuals are publicly available through Zenodo.

**Dataset**

https://doi.org/10.5281/zenodo.21326144

The dataset contains:

- PAN010 maternal haplotype assembly
- PAN010 paternal haplotype assembly
- PAN011 maternal haplotype assembly
- PAN011 paternal haplotype assembly
- PAN027 maternal haplotype assembly
- PAN027 paternal haplotype assembly
- PAN028 maternal haplotype assembly
- PAN028 paternal haplotype assembly
