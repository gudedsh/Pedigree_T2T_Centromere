# Interactive WashU Epigenome Browser Resource

To facilitate data sharing and visualization, all multi-generational pedigree assemblies and associated long-read epigenomic datasets generated in this study have been integrated into the **WashU Epigenome Browser**.

This directory contains browser data hubs, configuration files, and a step-by-step tutorial for loading and exploring the public datasets.

WashU Epigenome Browser:

https://epigenomegateway.wustl.edu/

---

## Available Resources

The browser resource includes:

- Pedigree T2T genome assemblies
- Gene, RepeatMasker, and CenSat annotations
- DNA methylation tracks
- Chromatin accessibility (Fiber-seq FIRE) tracks
- CENP-A DiMeLo-seq tracks
- Additional centromere epigenomic datasets described in this study

## Tutorial

The tutorial describes how to:

1. Open the WashU Epigenome Browser.
2. Select a pedigree T2T assembly as the reference genome.
3. Load public data hubs.
4. Visualize methylation, chromatin accessibility, and CENP-A datasets.
5. Navigate centromeric regions and customize browser tracks.
