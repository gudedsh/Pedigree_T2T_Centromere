# Interactive WashU Epigenome Browser Resource

To facilitate data sharing and maximize the reusability of our datasets, we have integrated our multi-generational pedigree long-read epigenomic assets into the **WashU Epigenome Browser**. This directory contains the complete browser data hubs, coordinate maps, track validation configurations, and a comprehensive step-by-step user tutorial.

WashU Epigenome Browser
https://epigenomegateway.wustl.edu/
